Machine Learning é uma área da inteligência artificial que combina estatística e computação.

Diferente da forma mais comum de usar um computador, dando instruções explícitas sobre a tarefa a ser executada, nós damos os dados ao computador e ele resolve o que fazer a partir dos padões desses dados.

Aqui estão as palavras-chave:

- **Algoritmos**: métodos matemáticos que aprendem com os dados.
- **Modelos**: resultado do treinamento dos algoritmos, usados para fazer previsões ou tomar decisões.
- **Dados de treinamento**: exemplos usados para ensinar o algoritmo.
- **Tipos de aprendizado**: supervisionado, não supervisionado, semi-supervisionado e por reforço.

## Tipos de aprendizado

- **Supervisionado**: O algoritmo recebe dados rotulados.
- **Não supervisionado**: O algoritmo busca padrões nos dados sem rótulos.
- **Semi-supervisionado**: O algoritmo recebe uma parte dos dados rotulados e busca padrões nos dados restantes.
- **Por reforço**: O algoritmo aprende por tentativa e erro.

## Principais algoritmos

- **Regressão Linear e Logística**: Usados para prever valores contínuos ou probabilidades.
- **Árvores de Decisão**: Estruturas que tomam decisões com base em perguntas sequenciais.
- **K-Nearest Neighbors (KNN)**: Classifica novos dados com base nos vizinhos mais próximos.
- **Redes Neurais**: Inspiradas no cérebro humano, são a base do deep learning.
- **Máquinas de Vetores de Suporte (SVM)**: Separação eficiente de classes em problemas de classificação.

## Etapas de um Projeto de Machine Learning

1. **Definição do probema**: Definir perguntas relevantes a serem respondidas a partir dos dados.
1. **Coleta e preparação dos dados**: Limpeza, normalização e seleção de atributos relevantes.
1. **Divisão dos dados**: Separação em conjuntos de treino e teste para avaliar o desempenho do modelo.
1. **Treinamento**: O algoritmo aprende com os dados de treino.
1. **Avaliação**: Teste do modelo com dados novos para medir sua precisão.
1. **Ajuste e otimização**: Melhorias no modelo para aumentar a performance.
